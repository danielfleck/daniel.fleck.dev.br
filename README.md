Os conhecimentos técnicos anteriores são uma base relevante, mas estão sendo **reabsorvidos e atualizados** por meio de estudo dirigido, projetos práticos, documentação, GitHub e portfólio. Cada item de portfólio deverá conter, quando aplicável:

- título;
- descrição;
- texto explicativo;
- imagem ou captura de tela;
- tecnologias utilizadas;
- link da página publicada;
